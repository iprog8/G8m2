About

Java 1

Java este un limbaj de programare orientat pe obiect, creat in 1995. Este detinut de Oracle si peste 3 milioane de dispozitive il utilizeaza.

HTML

HyperText Markup Language (HTML) este un limbaj pentru structurarea și prezentarea conținutului pentru World Wide Web, o tehnologie nucleu pentru Internet propusă inițial pentru browserul Opera.

MySQL

MySQL este un sistem de gestiune a bazelor de date relaționale, produs de compania suedeza MySQL AB.

JavaScript

JavaScript (JS) este un limbaj de programare orientat obiect bazat pe conceptul prototipurilor. E folosit mai ales pentru introducerea unor funcționalități în paginile web
